## Overview

Finance Pilot is a modern Android personal finance application designed to help you manage your money with clarity and confidence. It combines powerful budgeting and expense-tracking capabilities with a clean, premium user experience inspired by Apple's latest Liquid Glass design language.

Built using modern Android technologies, Finance Pilot offers comprehensive financial management while keeping your data private and under your control.

## Building the Project

### Requirements

* Android Studio (Latest Stable)
* JDK 17 or newer
* Android SDK
* Gradle

### Clone the Repository

```bash
git clone https://github.com/adon27/FinancePilot.git
```

### Open in Android Studio

Open the project folder and allow Gradle to synchronize.

### Build

```bash
./gradlew assembleDebug
```

### Run Tests

```bash
./gradlew test
```
